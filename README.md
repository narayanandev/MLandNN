

Navigate to your project root.

Sync the environment: Run uv sync to install dependencies into the .venv directory.

Activate (Optional): If you need to work directly in the shell, use the standard activation command for your OS:

macOS/Linux: source .venv/bin/activate

Windows: .venv\Scripts\activate 


Visual studio code will detect uv venv automatically when it opens a New terminal
